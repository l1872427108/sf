    # 图的深度优先遍历

    尽可能深的搜索图的分支。

    • 访问根节点。
    • 对根节点的没有访问过的相邻节点挨个进行深度优先遍历

    # 图

    const graph = {
    0: [1, 2],
    1: [2],
    2: [0, 3],
    3: [3]
}

    # coding

    const visited = new Set();
const dfs = (n) => {
    // 访问节点
    console.log(n);
    visited.add(n);
    graph[n].forEach(c => {
        if (!visited.has(c)) {
            dfs(c);
        }
    })
}
dfs(2);
